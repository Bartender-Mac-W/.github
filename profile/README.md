## Overview

Bartender Advanced is a menu bar management application that gives you complete control over the icons in your menu bar, allowing you to hide, rearrange, and organize status icons so that only the ones you need are always visible without a cluttered, overflowing status bar.

Hidden icons are accessible behind a single Bartender menu bar item that reveals your full set of status icons when clicked or triggered by a keyboard shortcut, keeping them available without consuming permanent menu bar real estate.

Show-on-update behavior makes specific icons visible automatically when they change state — so a download progress indicator appears when a download is in progress and hides when it completes — keeping transient information visible at the right moment.

Custom spacing, grouping, and ordering give you full control over the visual layout of visible menu bar icons, producing a clean, organized status bar that reflects your actual priorities rather than the default application install order.

For users with many menu bar applications whose combined icons overflow beyond the menu bar space, Bartender Advanced is the organization tool that restores order to the top of the screen permanently.

## Performance

Bartender balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
